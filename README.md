# KOC Growth Agent

面向普通 KOC 的纯前端 AI 增长工作台 Demo。作品覆盖「画像诊断 → 选题雷达 → 内容工坊 → 发布日历 → 增长复盘」完整闭环，默认不依赖后端，适合部署到 GitHub Pages。

## 运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建产物在 `dist/`，可直接用于 GitHub Pages。项目使用 `HashRouter` 和相对资源路径，避免 GitHub Pages 子路由刷新 404。

## 评审体验路径

1. 打开 Demo 后进入「诊断」页。
2. 点击「使用示例账号」生成 KOC 画像诊断。
3. 进入「选题雷达」，生成并筛选内容机会。
4. 选择选题进入「内容工坊」，生成标题、脚本、分镜、封面文案、标签和评论话术。
5. 进入「发布日历」，生成 7 天排期。
6. 进入「增长复盘」，查看模拟数据与下一轮优化建议。

## 初赛说明文档结构

可打开 `docs/submission.html`，用浏览器打印为 PDF。

- 作品名称：KOC Growth Agent
- 赛道：AI + 社媒流量密码，帮普通 KOC 轻松涨粉
- 目标用户：普通 KOC、新手博主、小商家内容运营者
- 核心痛点：定位模糊、选题低效、内容生产不稳定、不会复盘、商业化路径不清楚
- 产品方案：用 AI Agent 串联诊断、选题、生成、排期和复盘
- AI 原生性：AI 不是单点文案生成，而是贯穿增长任务拆解和持续优化
- 商业化思考：订阅制、账号诊断报告、品牌合作撮合、内容模板市场
