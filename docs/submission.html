KOC Growth Agent

赛道：AI + 社媒流量密码，帮普通 KOC 轻松涨粉

一个面向普通 KOC、新手博主和小商家内容运营者的 AI 增长工作台，把账号定位、内容选题、脚本生产、发布排期和数据复盘串成可执行闭环。

目标用户与痛点

普通 KOC知道自己想做内容，但缺少定位方法和持续选题能力。
新手博主会写单条内容，但不知道如何形成账号记忆点。
小商家运营者需要低成本做内容获客，但不会把产品自然植入场景。
核心痛点定位模糊、选题低效、内容生产不稳定、不会复盘、商业化路径不清楚。

产品方案

KOC Growth Agent 不是单点文案生成器，而是一个增长 Agent：先判断目标用户和内容定位，再生成选题、脚本和排期，最后基于数据表现给出下一轮优化动作。

画像诊断 选题雷达 内容工坊 发布日历 增长复盘

核心功能

画像诊断输入领域、平台、账号阶段和目标用户，生成定位、人设、机会点和风险提醒。
选题雷达按用户匹配、爆款潜力、制作难度和商业价值评估选题。
内容工坊生成标题、开头钩子、短视频脚本、分镜、封面文案、标签和评论互动话术。
发布日历规划 7 天内容节奏，明确平台、发布时间、主题和素材状态。
增长复盘用播放、点赞、评论、收藏、转粉数据输出诊断和下一轮动作。
设置默认 Mock AI 可稳定体验，可选 API Key 模式，本地保存、不依赖后端。

AI 原生能力

AI 贯穿任务拆解、场景匹配、内容策略、执行计划和结果优化。产品让低经验 KOC 不需要理解复杂运营框架，也能按 Agent 提供的步骤完成一次内容增长循环。

创新与差异化

相比普通 AI 写作工具，本作品强调「增长工作流」而非「单条文案」。选题会被结构化评分，脚本会绑定平台和用户场景，复盘会反向生成下一轮内容池。

落地可行性与商业化

初赛版本为纯前端静态 Demo，可部署到 GitHub Pages。后续可接入真实模型 API、平台数据授权、内容发布 API、账号诊断报告和品牌合作撮合模块。商业模式包括订阅制、诊断报告、模板市场和品牌商单匹配。
