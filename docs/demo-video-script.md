# KOC Growth Agent 3 分钟演示视频脚本

## 0:00-0:30 产品开场

普通 KOC 最大的问题不是不会写一条文案，而是不知道自己该面向谁、持续做什么内容、怎么复盘优化。KOC Growth Agent 是一个 AI 增长工作台，把诊断、选题、生成、排期和复盘连成完整闭环。

## 0:30-1:00 画像诊断

打开 Demo 后直接进入工作台。点击「使用示例账号」，系统基于领域、平台、目标用户和痛点生成账号人设、内容定位、机会方向和风险提醒。

## 1:00-1:40 选题雷达

进入选题雷达，AI 生成多条内容机会，并用用户匹配、爆款潜力、制作难度和商业价值进行评分。评委可以按平台筛选，并选择一条选题进入下一步。

## 1:40-2:20 内容工坊

内容工坊会生成标题组、前三秒钩子、短视频脚本、分镜、封面文案、标签和评论区互动话术。这里体现 AI 不只是写文案，而是在帮助 KOC 把选题变成可拍、可发、可互动的内容资产。

## 2:20-2:45 发布日历与复盘

发布日历生成 7 天内容节奏，明确平台、发布时间、内容主题和素材状态。增长复盘页用模拟数据展示播放、点赞、评论、收藏和转粉表现，并生成下一轮优化建议。

## 2:45-3:00 收尾

KOC Growth Agent 面向普通 KOC 的真实增长痛点，用 AI Agent 串联从策略到执行再到复盘的全过程。初赛版本无需后端即可部署，后续可接入真实平台数据、模型 API、账号诊断报告和品牌合作能力。
